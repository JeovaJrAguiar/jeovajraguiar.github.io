Guia de Uso: LaTeX no seu GitHub Pages

Este documento demonstra como usar sintaxe LaTeX para criar fórmulas matemáticas e melhorar a apresentação acadêmica do seu site.

Fórmulas Inline

Para inserir fórmulas no meio do texto, use $formula$. Por exemplo: A equação mais famosa da física é $E = mc^2$, onde $E$ é a energia, $m$ é a massa e $c$ é a velocidade da luz.

Outros exemplos: $\alpha + \beta = \gamma$, $\sum_{i=1}^{n} x_i$, $\int_0^1 f(x)dx$.

Fórmulas em Bloco

Para fórmulas em destaque, use $$formula$$:

Identidade de Euler

$$e^{i\pi} + 1 = 0$$

Fórmula Quadrática

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Integral Gaussiana

$$\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}$$

Equação de Schrödinger

$$i\hbar\frac{\partial}{\partial t} \Psi(\mathbf{r},t) = \hat{H} \Psi(\mathbf{r},t)$$

Matrizes e Sistemas

Exemplo de matriz:

$$\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx + dy \end{pmatrix}$$

Sistema de equações:

$$\begin{cases} x + y = 5 \\ 2x - y = 1 \end{cases}$$

Símbolos Comuns em Criptografia

Aqui estão alguns símbolos úteis para suas pesquisas em criptografia:

  • Operações: $\oplus$ (XOR), $\land$ (AND), $\lor$ (OR), $\neg$ (NOT)
  • Conjuntos: $\mathbb{Z}$ (inteiros), $\mathbb{R}$ (reais), $\mathbb{C}$ (complexos)
  • Criptografia: $\text{Enc}_k(m)$ (encriptação), $\text{Dec}_k(c)$ (decriptação)
  • Probabilidade: $\Pr[X = x]$, $\mathbb{E}[X]$ (valor esperado)
  • Complexidade: $O(n)$, $\Theta(n)$, $\Omega(n)$

Exemplo: Segurança Semântica

Um esquema de criptografia é semanticamente seguro se:

$$\Pr[\mathcal{A}(\text{Enc}_k(m_0)) = 1] - \Pr[\mathcal{A}(\text{Enc}_k(m_1)) = 1] \leq \text{negl}(\lambda)$$

Como Usar

Para usar LaTeX no seu site:

  1. Inclua o MathJax no <head> de suas páginas HTML
  2. Use $formula$ para fórmulas inline
  3. Use $$formula$$ para fórmulas em bloco
  4. Use comandos LaTeX padrão: \frac{}{}, \sum, \int, etc.
Dica: Este formato é perfeito para artigos acadêmicos, papers, e documentação técnica. O estilo visual imita documentos LaTeX tradicionais, dando um aspecto profissional e acadêmico ao seu site.
